A machine learning-based web application for predicting heart disease risk using the CDC BRFSS 2015 dataset. This project achieves 81.7% recall, successfully identifying over 4 out of 5 patients with heart disease.

#Project Overview
This project was developed as part of the Introduction to Data Science (DL2001) course at FAST-NU Lahore by:

#Technologies Used:
Python 3.8+
Streamlit - Web interface
scikit-learn - Machine learning models
pandas - Data manipulation
numpy - Numerical computing
pickle - Model serialization

#Key Features
✅ Interactive web-based prediction interface
✅ Random Forest model with 81.7% recall
✅ Real-time risk assessment
✅ Comprehensive feature analysis
✅ User-friendly Streamlit dashboard

#Dataset Information
Source: CDC Behavioral Risk Factor Surveillance System (BRFSS) 2015
Total Samples: 253,680 health survey responses
Features: 22 health and demographic indicators
Target: Heart Disease or Attack (Binary Classification)
Class Balance: Handled via undersampling (50-50 split)
